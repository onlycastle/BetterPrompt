# Code of Conduct

## Expected Behavior

- Be respectful and specific.
- Assume good intent, but keep technical discussion rigorous.
- Focus feedback on the code and the behavior, not the person.

## Unacceptable Behavior

- Harassment, discrimination, or intimidation
- Publishing private information without consent
- Disruptive or abusive conduct in issues, discussions, or pull requests

## Enforcement

Project maintainers may remove content or restrict participation for behavior that violates this code of conduct.
